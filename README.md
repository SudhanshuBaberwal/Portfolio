## ⚙️ Installation & Setup

1. Clone the repository  
```bash
git clone https://github.com/SudhanshuBaberwal/Portfolio.git
```

2. Navigate to project folder  
```bash
cd Portfolio
```

3. Install dependencies  
```bash
npm install
```

4. Run the project  
```bash
npm run dev
```
